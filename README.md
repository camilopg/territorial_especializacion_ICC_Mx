# territorial_especializacion_ICC_Mx
# Reproducible script for the territorial specialization chapter of the ICC in 4 urban nodes of Mexico
# Descripción
Este repositorio contiene el código y los procedimientos utilizados para analizar la especialización territorial de las industrias culturales y creativas (ICC) en cuatro nodos urbanos mexicanos:
  •	Área Metropolitana de Guadalajara (AMG)
  •	Puerto Vallarta
  •	Oaxaca
  •	Tijuana

El estudio utiliza establecimientos económicos georreferenciados del DENUE (INEGI) clasificados de acuerdo con el marco conceptual de la Cuenta Satélite de la Cultura de México (CSCM)

# Pregunta de investigación
¿Cómo se especializan las industrias culturales y creativas en los principales nodos urbanos mexicanos?

# Métodos
  •	Estadística descriptiva
  •	Distribuciones sectoriales
  •	Índices de Especialización Territorial
  •	Cocientes de Localización (LQ)
  •	Chi-cuadrada

# Datos
•	DENUE 2026 (Jalisco, Baja California y Oaxaca)
•	AGEB urbanas INEGI
